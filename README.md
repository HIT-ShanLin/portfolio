# 个人主页 / Personal Portfolio

这是我的个人简介网站，展示工作经历、技术能力和 GitHub 项目。使用 [Astro](https://astro.build) 构建，托管于 GitHub Pages。

This is my personal portfolio website showcasing work experience, technical skills, and GitHub projects. Built with [Astro](https://astro.build), hosted on GitHub Pages.

🌐 **[hit-shanlin.github.io/portfolio](https://hit-shanlin.github.io/portfolio/)**

## 特性 / Features

- 🌍 中英双语 / Bilingual (Chinese & English)
- 🌓 深色/浅色主题 / Dark & Light theme
- 📂 GitHub 项目自动拉取 / Auto-fetched GitHub projects
- 📱 响应式设计 / Responsive design

## 本地开发 / Local Development

```bash
npm install
npm run dev
```
